# 证券行业问句匹配数据集

该数据集主要用于证券行业自动问答系统语料，共包含11个问题类别，2万多条问句，其中每一个问句都标注了所属类别。数据集中存在错别字、语法不规范等问题，但更贴近实际应用场景。

示例：

概念 [分隔符] 波导股份是什么样的题材股？

交易所板块 [分隔符] 华扬联众属于哪个交易所板块？

趋势 [分隔符] 方盛制药这个票中线怎么样

......
